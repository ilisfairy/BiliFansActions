# 📈 BiliFansAction

BiliFansAction 是一个基于 Python 的工具，利用 GitHub Actions 每日自动更新 Bilibili UP 主的粉丝数变化并绘图。该工具可以帮助 Bilibili UP主更加直观地了解自己的粉丝变化情况。

### 依赖

在使用 BiliFansAction 之前，需要先安装以下依赖：

- Python 3.9
- Matplotlib 3.3.4
- Requests
- pandas

然后 clone 本仓库，最后自己折磨怎么用吧（）
